## 🍼 baby-dev

> CUITBCA区块链协会新成员的学习与作业仓库。
> 用于提交代码作业、学习笔记、项目练习
> 让每一位成员都能在链上留下自己的成长足迹。

---

### 📘仓库结构

```
baby-dev/
│
├── week1/                # 第一周开发任务
│   ├── Alice/            # 成员个人文件夹
│   │   ├── homework1.md  # 作业信息或笔记
│   │   └── code/         # 可选：代码目录
│   │       └── hello_move.sol
│   └── Bob/
|   └──README.md          # 每周任务说明
│
└── README.md
```

---

### 📥提交规范

1. **Fork 仓库** 到你自己的 GitHub 账户。
2. **新建文件夹**：在对应周次下建立你的个人文件夹，例如：

   ```
   week2/Alice/
   ```
3. **放入作业/周报**：

   * 笔记命名：`<笔记主题>_weekX.md`
   * 代码文件可单独放在 `code/` 子目录中。
4. **提交 Commit 信息规范**：

   ```
   git commit -m "week2: 完成ERC20学习"
   ```
5. **发起 Pull Request**：标题示例

   ```
   [Week2] Alice 提交作业
   ```

---

### 🧩命名规范

| 类型    | 说明          | 示例                              |
| ----- | ----------- | ------------------------------- |
| 周次文件夹 - 由仓库管理员决定 | 对应第几周       | `week1`, `week2`                |
| 成员文件夹 | 英文昵称        | `Alice`, `Bob`                  |
| 笔记文件  | 主题 + 周次 | `ERC20_week3.md`               |

> ⚠️ 请勿使用中文路径或空格，避免出现合并冲突或文件识别问题。
